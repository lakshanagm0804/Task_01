# Task 1: Data Cleaning and Preprocessing

## Internship Details

**Domain:** Data Analytics

## Objective

The objective of this task was to clean and preprocess a raw dataset to improve data quality and prepare it for further analysis.

## Dataset

Netflix Dataset

## Tools Used

* Python
* Pandas
* Jupyter Notebook

## Steps Performed

### 1. Data Loading

* Imported the dataset into a Pandas DataFrame.

### 2. Data Inspection

* Checked dataset shape, columns, and data types.
* Identified missing values.

### 3. Duplicate Removal

* Detected and removed duplicate records.

### 4. Missing Value Handling

* Filled missing values in relevant columns.
* Removed rows with critical missing information where necessary.

### 5. Data Type Modification

* Converted columns to appropriate data types.
* Converted date columns into datetime format.

### 6. Data Standardization

* Standardized column names for consistency.

### 7. Saving the Cleaned Dataset

* Exported the cleaned dataset as a new CSV file.

## Outcome

Successfully cleaned and preprocessed the dataset by handling missing values, removing duplicates, modifying data types, and preparing the data for further analysis.
